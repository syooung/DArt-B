# 1️⃣ 개념 정리 
## 01. 인과와 상관

```
✅ 학습 목표 :
* 인과관계와 상관관계의 차이점에 대해 설명할 수 있다.
* 인과관계와 상관관계의 다양한 사례를 읽고 이해한다.
```
<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
### 인과관계
**원인과 결과의 관계** 를 뜻함  
즉, 원인을 바꾸면 결과가 달라지는 관계임   
**원인 -> 결과** 라는 방향성을 가짐

### 상관관계
데이터에서 보이는 **관련성(association)** 을 말함
어떤 특정한 조합이 일어나기 쉽고, 확률변수 사이가 독립이 아니라는 것을 뜻함

### 인과와 상관의 차이
> ex. 아침밥을 먹는 빈도와 학교 성적의 관계
>> “아침밥을 잘 챙겨 먹으면 성적이 오른다“ 는 가설  
>> **관찰연구** : 실험 없이 현재 상태를 관찰  
>> **실험연구** : 각 학생에게 아침밥을 먹는 빈도를 무작위로 할당한 개입 실험을 수행하고, 그 후의 성적을 조사  
>> **`무작위 통제 실험`** -> 인과관계 주장 가능

**`중첩`** : 두 변수에 관련된 외부 변수가 존재할 때, 이를 ***중첩***이라 함.  
그 변수를 **중첩요인(중첩변수, confounder)** 이라 함

### 허위상관
**인과관계는 없지만 상관관계는 있을 때**

### 인과관계를 알면 할 수 있는 일
*더 많은 것 이해 가능*  
**개입**이 가능함 => 원인변수를 변화시킴으로써(개입), 결과 변수를 바꿀 수 있음

### 상관관계를 알면 할 수 있는 일
**예측**이 가능함 -> 상관관계는 2개 변수 X, Y 사이의 관련성이므로, 한쪽 변수로부터 또 다른 변수를 예측할 수 있음

### 인과관계와 상관관계의 다양한 사례

* 허위상관의 예  
'아이스크림 매출과 수영장 익사 사고 사이의 양의 상관관계'  
이는 중첩요인인 *기온*으로 상관관계 발생 (인과관계 X)

* 시간, 나이는 중첩요인이 되기 쉬움  
*몸은 나이가 들수록 쇠하고, 기온은 온난화로 인해 해마다 높아짐*  
-> 즉 시간에 따른 “현실”이 상관관계로 나타난 것   
-> 따라서 알고자 하는 변수에 *시간 변수*가 관련되어 있지 않은지 주의
* 초콜릿 소비량과 노벨상 수상자 수  
'초콜릿 소비량과 노벨상 수상자 간 양의 상관관계'   
이는 중첩요인인 *GDP* 로 상관관계 발생 (인과관계 X)
* 우연히 생긴 상관  
수많은 변수를 마구잡이로 해석하면 통계적으로 유의미한 결과를 얻을 수도 있다는 문제 발생

## 02. 무작위 통제 실험

```
✅ 학습 목표 :
* 무작위 통제 실험에 대해 설명할 수 있다.
```

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
인과관계 발견이 어려운 이유 중 하나는 **중첩요인의 존재** 때문

음주가 폐암에 미치는 효과를 알고자 한다면, ***음주 이외의 요인을 동일하게  적용해야 함***

### 무작위 통제 실험(randomized control trial, RCT)
: 알고자 하는 요인인 변수 X에 표본을 무작위로 할당하고 개입 실험을 수행한 다음, 변수 Y와 비교하는 방법

> 왜 무작위 통제 실험이 인과효과를 추정하는 강력한 방법?
>> 중첩요인을 확인하지 않더라도, 그 효과를 무작위를 이용하여 ***무효화***할 수 있으므로, **알고자 하는 변수의 효과만 추정 가능하기 때문**

### 인과 추론의 근본 문제
조치를 취한 세계와 취하지 않은 세계 양쪽을 관찰할 수 없기 때문에 **현실에서 정확한 인과관계로 인한 차이를 측정할 수 없음**

=> 개인 수준이 아닌 집단 수준을 생각하여, ***인과의 평균적인 효과*** 고려

$$\tau' = E[Y^{(1)} \mid X = 1] - E[Y^{(0)} \mid X = 0]$$

### 선택평향
관측가능한 $\tau’$ 는 원래 알고자 하는 효과 $\tau$ 에 편향이 더해진 값이 됨  
**`중첩요인`** 이 선택편향을 발생시켰기 때문에 인과관계를 밝혀 내기가 어려웠던 것

## 03. 통계적 인과 추론
```
✅ 학습 목표 :
* 통계적 인과 추론을 이해한다.
* 다중회귀, 층별 해석, 경향 점수 짝짓기, 이중차분법 등의 방법을 이해한다.
```
<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
### 통계적 인과 추론
: 실제 데이터에서 인과효과를 추정하는 것은 쉬운 일이 아님  
: 많은 비용이 들거나, 큰 표본을 모으기가 쉽지 않음  
=> 통계적 인과 추론이라는 인과 효과의 추정 방법 사용

### 다중회귀
원인변수를 x, 결과변수를 y, 중첩요인을 z를 설명변수로 추가  
 $y= a + b_1x + b_2z$  
`조정`: 중첩요인을 포함하여 측정하는 것

### 층별 해석
중첩요인을 기준으로 데이터를 몇 가지 그룹(층)으로 나누어, 각 층 안에서 중첩요인의 효과를 ***가능한 한 작게*** 하는 방법  
ex. 성별, 나이 등으로 나누어 각각 해석

> `다중회귀` 에서는 각각의 설명변수가 독립으로 인과효과를 갖는다고 가정  
> **`층별해석`** 에서는 **층마다 다른 인과효과**를 추정할 수 있음

### 경향 점수 짝짓기 (Propensity Score Matching, PSM)
`원인변수=0`인 집단과 `원인변수=1`인 집단에서 비슷한 중첩요인을 가진 데이터를 골라 쌍으로 만드는 것

**경향 점수** => `원인변수=1` 에 할당되는 확률을 나타냄

```
<경향점수를 구하는 구체적인 순서>

	1.	먼저 반응변수를 원인변수(0 또는 1)로 하고,
	2.	중첩요인을 설명변수로 한 로지스틱 회귀 실행
	3.	이에 따라 어떤 중첩요인을 `원인변수=1`에 할당할 것인가를 평가
	4.	 고른 쌍으로 반응변수의 차이를 계산하고, 그 평균값을 취해 효과추정량으로 삼음
```

### 이중차분법(Difference In Differences, DID)
시간 축을 도입, 집단 간 차이에 대해 다시 한번 처리 전후의 차분을 취함으로써 인과효과를 추정


<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
### 통계학의 2가지 흐름
**`빈도주의 통계`**  
**`베이즈 통계`** : 컴퓨터를 사용함으로써 복잡한 통계 모형도 추정 가능

### 불확실성 다루기
: **빈도주의** 흐름에서의 불확실성은, 모집단에서 **표본을 추출할 때의 불확실성**임

**`빈도주의`** 에서 **확률**은 무한히 반복 실행한 결과로서의 객관적인 빈도를 나타냄  
**`베이즈 통계`** 는 **확률**을 ‘얼마나 확신하는지’로 해석하며, 데이터 획득을 통해 파라미터 확률분포의 불확실성이 줄어드는 모습을 보임

### 베이즈 통계의 이미지
: 데이터를 알기 전에는 파라미터에 대한 정보가 없으므로 모든 값이 같은 정도로 나타난다는 **균등분포**를 가정함  
: 데이터를 얻게 되면 특정 값 부근일 가능성이 커지면서 확률분포가 달라짐

### 통계 모형

통계모형의 목적은 **모집단의 실제 분포**를 아는 것임  
But, 실제 모집단 분포 $q(x)$를 직접 알 수 없으므로, 데이터로부터 이를 추측하기 위해 가정한 확률분포 $p(x|\theta)$를 의미함.  
: 파라미터 $\theta$에 구체적인 값을 지정하면 확률분포의 형태가 하나로 결정됨. 

### 최대가능도 방법
: 빈도주의 추정법으로, 얻은 데이터가 발생할 가능성, 즉 **가능도 L(\theta)** 를 최대화하는 파라미터 $\hat{\theta}$를 탐색함.   
=> **최대가능도 추정량**  
: 파라미터 추정 결과를 분포가 아닌 하나의 **'점'** 으로 얻는 것이 특징임.

### 베이즈 통계의 사고방식
: 통계 모형의 파라미터 $\theta$ 를 확률변수로 취급  
**사전분포** : 분석자가 데이터를 알기 전 단계의 $\theta$ 확률분포  
**사후분포** : 데이터를 안 후의 파라미터 $\theta$ 확률분포

**베이즈 정리**

$$
    p(\theta \mid x_1, x_2, \ldots, x_n)
    =
    \frac{
    p(x_1, x_2, \ldots, x_n \mid \theta)\, p(\theta)
    }{
    p(x_1, x_2, \ldots, x_n)
    }
$$

베이즈 통계는 **사전분포와 가능도에서 사후분포를 구하는 것**이 목적임

### 사전분포
: 데이터 x를 얻기 전에 파라미터 $\theta$ 가 어떤 분포인가를 미리 실험자나 해석자가 설정해야 하는 분포

But, 이 분포를 연구자의 주관적인 판단으로 결정한다는 점에서 비판 받음  
=> 따라서 **균등분포**나 **정규분포**와 같이 어떤 정보도 가지지 않을 법한 분포를 사전분포로 가정

**균등분포** : 최솟값과 최댓값 사의 모든 값이 같은 정도로 발생하는 분포  
=> 아무정보를 갖지 않음

### 베이즈 추정의 예측분포
: 베이즈 추정으로 얻은 파라미터 $\theta$ 의 사후분포로 예측분포를 만들 수 있음
$$p^*(x) = \int p(x|\theta) p(\theta|x_1, x_2, \dots, x_n)$$

### 정보량 기준
`쿨백 —라이블러 발산` : 실제 모집단분포와 예측분포가 어느정도 일치하는가를 평가  
=> 값이 작을수록 실제 모집단분포를 잘 나타내며, 새롭게 얻을 데이터를 예측하는 성능도 높아짐

**WAIC(Widely Applicable Information Criteria)** : 베이즈 추정에서 사용. 복잡한 모형에서도 사용가능

### MCMC 방법 (Markov Chain Monte Carlo method)
: 특정 확률분포를 따르는 난수 발생 알고리즘

베이즈 통계에서는 이것을 이용해 사후분포를 따르는 난수를 발생시키고, 그 난수의 집합을 관찰함으로써 사후분포의 성질을 분석

1.	**몬테카를로 방법** : 난수를 여러 개 발생시켜 시뮬레이션해 근사해를 얻는 방법

2.	**마르코프 연쇄**: 어떤 상태에서 다른 상태로 변화하는 현상을 확률로 표현한 모형의 일종

3.	**깁스 표집(Gibbs samping)**: 구체적인 계산은 한쪽 변수를 고정한 뒤, 고정하지 않은 변수를 확률적으로 움직이는 작업을 번갈아 반복하는 순서로 이뤄짐

**burn-in**, **warm-up** : 초기 상태에서 수렴하기까지의 기간은 버려야 함

## 06. 베이즈 통계 사례

```
✅ 학습 목표 :

```

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
### 이표본 평균값 비교

신약과 위약을 투여한 두 집단의 평균 혈압 차이를 베이즈 통계로 해석한 사례  
: 사전분포로 0~300의 균등분포를 설정하고, MCMC 방법의 처음 1,000단계는 버리는 기간인 *번인*으로 간주한 뒤 나머지 단계를 사후분포 분석에 사용함.  
: 사후분포를 요약하는 점 추정값으로 평균값인 **사후기댓값(EAP)** 이나 최빈값인 **사후최빈값(MAP)** 을 계산함.   
: 폭을 이용한 추정으로는 파라미터가 해당 범위 안에 있을 확률을 나타내는 **95% 신뢰구간(CI, 확신구간)** 을 구하며, 이는 빈도주의의 신뢰구간과는 의미가 다름.  
=> 이를 통해 "2개 집단의 평균값 차이가 -10 이하가 될 확률은 98.7%"와 같이 직관적이고 정량적인 논의가 가능해짐

### 푸아송 회귀의 예

: 일반화선형모형(GLM)인 푸아송 회귀를 베이즈 통계로 추정해 절편항 a와 기울기 b의 사후분포를 각각 얻을 수 있음  
: 사전분포를 **균등분포**로 설정하고, MCMC 결과를 통해 EAP 및 95% 신뢰구간을 도출함.  
=> 얻어진 기울기 파라미터 b의 95% 신뢰구간에 주목하면, 설명변수가 반응변수에 미치는 효과를 정량적으로 파악하여 모형을 심도 있게 검토할 수 있음.

### 계층적 베이지안 모형
개체 차이의 사전분포의 형태를 결정하는 파라미터 s 가 있고, 이 s 에 관해서 사전분포가 설정되었을 때, **계층적 사전분포**라고 부름  
-> 이 개체 차이 파라미터의 분포 형태를 결정하는 파라미터를 **초모수(hyper-parameter)** 라 부름

=> 이처럼 파라미터에 대한 사전분포가 계층적으로 설정된 계층적 사전분포 방식을 사용하는 모형을 계층적 베이지안 모형이라고 함.
