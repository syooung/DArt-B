히스토그램만으로는 데이터를 *객관적*이고 *정략적*으로 평가, 서술할 수 없음  
### 통계량
: 수집한 데이터로 이런저런 계산을 수행하여 얻은 값  

**기술통계량(요약통계량)**  
: 데이터 그 자체의 성질을 기술하고 요약하는 통계량  
: ex. 평균값  
: 주로 양적 변수를 대상으로 함  

```
<대표적인 기술통계량>
- 대푯값: 평균값, 중앙값, 최빈값
- 데이터 퍼짐 정도를 나타내는 값: 분산, 표준편차 
```
### 대푯값(representive value)
: 대략적인 분포 위치, 즉 대표적인 값을 정량화하기 위해 사용하는 통계량

**1. 평균값(mean)**
- 가장 자주 쓰이며, 잘 알려진 대푯값
- '표본평균'이라고도 함  

    $\bar{x}$ = $\frac{1}{n}$($x_1$+$x_2$+...+$x_n$) = $\frac{1}{n}$ $\sum_{i=1}^{n} x_i$ 


**2. 중앙값(median)**
- 크기 순으로 값을 정렬했을 때 한가운데 위치한 값

- 표본크기 n이 짝수라면, 가운데 있는 값 2개의 평균값을 중앙값으로 함

- 순서에만 주목하기에, 극단치가 있어도 영향을 받지 않음

**3. 최빈값(mode)**
- 데이터 중 가장 자주 나타나는 값

- 연속변수일 떄는 히스토그램처럼 범위를 정한 뒤, 그 사이에 들어가는 데이터 개수로 정함


### 대푯값의 모습
$\rightarrow$ 분포가 좌우대칭에 가까운 봉우리 형태라면 평균값, 중앙값, 최빈값은 대체로 일치함.  
$\rightarrow$ 좌우 비대칭 분포라면 일치하지 않음  
 ex. 오른쪽/왼쪽 꼬리가 긴 형태

**대푯값을 통해 분포의 위치를 정량적으로 특징 지을 수 있음**

**이상값(outlier)**  
: 드물게 극단적으로 큰 값이나 작은 값  
: 일반적으로 평균값에서 표준편차의 2배 또는 3배 이상 벗어난 숫자 

$\rightarrow$ 평균값은 이상값의 영향을 받기 쉬움  
$\rightarrow$ but, 중앙값은 가운데에 있는 값만 참조하므로 이상값에는 잘 영향받지 않음   

**히스토그램의 중요성**  
대푯값은 편리하지만, 분포가 봉우리 형태가 아닌 경우, 왜곡된 값을 얻을 수도 있음  
따라서 **히스토그램**을 그려 대략적인 파악을 하는 게 중요.

### 분산과 표준편차
: 분포의 폭, 즉 데이터가 '어느 정도 퍼져 있는지(흩어져 잇는지)'를 파악  

**1. 표본분산(variance)**
- 표본의 각 값과 표본평균이 어느 정도 떨어져 있는지를 평가하는 것
- 데이터 퍼짐 상태를 정량화한 통계량  

$$s^2 = \frac{1}{n}\{(x_1-\bar{x})^2 + (x_2-\bar{x})^2 + \cdots + (x_n-\bar{x})^2\} = \frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2$$

<표본분산의 성질>

* $s^2 \geqq 0$  
* 모든 값이 같다면 0
* 데이터 퍼짐 정도가 크면 $s^2$이 커짐  

**2. 표본표준편차(standard deviation, S.D)**

- 표본분산의 제곱근을 취한 값

$$s=\sqrt{s^2}=\sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2}$$

**상자수염그림(box-and-whisker plot)**

- 상자와 수염으로 구성되며, 각각은 데이터의 분포를 특징 짓는 통계량을 나타냄
- 범위의 최댓값/최솟값 : 상자 길이 * 1.5
- 제1사분위수 / 제2사분위수 / 제3사분위수

**분포를 시각화하는 다양한 방법**
: 오차막대 / 바이올린 플롯 / 스웜 플롯

```
✅ 학습 목표 :
```

**확률** : (발생 여부가) 불확실한 사건의 발생 가능성을 숫자로 표현한 것  
$\rightarrow$ 확률은 영문자  P로 표기하므로, 사건 A의 확률은 P(A)로 표기

**확률변수** : 확률이 달라지는 변수  
**실현값** : 확률변수가 실제로 취하는 값

**확률분포** : 가로축에 확률변수를, 세로축에 그 확률변수의 발생 가능성을 표시한 분포  

>범주형 변수, 이산형 양적 변수인 경우  
>> $\rightarrow$ 가로축은 확률변수, 세로축은 확률


**확률밀도함수** : 확률변수가 연속형일 때 일정한 범위를 두고 확률을 구하는데, 그 확률을 계산하는 함수

### 추론통계의 확률분포

다루기 어려운 ‘모집단과 표본 데이타’를 다룰 수 있는 ’확률분포와 그 실현값‘으로 치환

### 기댓값(expected value)
: ’변수가 확률적으로 얼마나 발생하기 쉬운가‘를 평균적인 값으로 나타낸 것

### 분산과 표준편차

**표준편차**는 분산 V(X)의 제곱근을 취한 값
~~~
	⁃	0 이상일 것
	⁃	모두 같은 값이 나타나는 경우에는 0
	⁃	기댓값에서 떨어진 값이 많을수록 커짐
~~~

### 왜도와 첨도
왜도(skewness) : 분포가 좌우대칭에서 어느 정도 벗어났는지
첨도(kurtosis) : 분포가 얼마나 뾰족한지, 그래프의 꼬리가 차지하는 비율이 얼마인지

### 확률변수가 2개일 때
**동시확률분포 P(X, Y)** : 확률변수 2개
 
**독립** 
P(X, Y) = P(X) x P(Y)

**조건부확률**
확률변수 Y의 정보가 주어졌을 때, 다른 한쪽 확률변수 X의 확률을 조건부확률 $$P(A|B)$$ 


## 05. 이론적인 확률분포

```
✅ 학습 목표 :
* 모수의 개념과 통계적 의미를 이해한다.
* 정규분포의 특성과 표준화의 개념을 설명할 수 있다.
```

### 파라미터(parameter, 모수)
: 이론적인 확률분포는 수식으로 표현되며, 분포의 형태를 정하는 파라미터를 가짐

*데이터 분석의 목적은 **모집단의 성질을 아는 것***
‘~ 이라는 파라미터를 가진 ~이라는 확률분포’로 나타낼 수 있음
—> 이는 모집단의 성질을 아는 것이 됨

### 정규분포 (가우스 분포)
확률분포는 **평균 $\mu$ 와 표준편차 $\sigma$** 라는 2개의 파라미터로 정해짐
평균  $\mu$=0, 표준편차 $\sigma$=1 인 정규분포 N(0,1)을 **표준정규분포**라 함

**정규분포의 특징**  

⁃	평균 $\mu$ 를 중심으로 한 종형으로, 좌우대칭 분포이다.  
⁃	평균 $\mu$ 근처에 값이 가장 많고, 평균 $\mu$에서 멀어질수록 적어진다.  
⁃	키나 몸무게 등 정규분포로 근사할 수 있는 현상이 많다

**정규분포의 성질**

* $\mu - \sigma$ 부터 $\mu + \sigma$ 까지의 범위에 값이 있을 확률은 약 68%
* $\mu - 2\sigma$ 부터 $\mu + 2\sigma$ 까지의 범위에 값이 있을 확률은 약 95%
* $\mu - 3\sigma$ 부터 $\mu + 3\sigma$ 까지의 범위에 값이 있을 확률은 약 99.7%

### 표준화(standardizing, normalizing), z값

$$z = \frac{x - \mu}{\sigma}$$
평균과의 거리가 표준편차의 몇배인가를 나타내기 때문에, 본래의 $\mu$나 $\sigma$와 상관없이 ***분포 안에서 어디에 위치하는가***를 알 수 있음



## 06. 추론통계를 배우기 전에

```
✅ 학습 목표 :
* '데이터를 얻는다는 것'의 의미를 이해한다.
* 무작위추출의 개념과 통계적 필요성을 설명할 수 있다.
* 추론통계의 직관적 의미를 이해한다
```

**전수조사** : 모집단의 모든 요소를 조사  
**표본조사** : 표본(모집단의 일부)으로 모집단의 성질을 추정

### 데이터(표본)를 얻는다는 것
모집단에 포함된 전체 값으로 구성된 분포에서 일부를 추출하는 것

**"모집단=확률분포"** / **"표본=확률분포를 따르는 실현값"**

### 데이터로부터 그 발생원의 확률분포 추정하기
“얻은 표본으로 모집단을 추정한다” $\rightarrow$ “얻은 실현값으로 이 값을 발생시킨 확률분포를 추정한다”

### 모집단분포 모형화(modeling)
: 수학적인 확률분포로 모집단분포를 근사하는 것  
: **수식**으로 기술하면 수학적으로 다룰 수 있는 확률분포에 근사하여 모집단의 추정이 용이해짐

### 무작위추출(random sampling)
: 데이터를 얻을 때 모집단에 포함된 요소를 하나씩 무작위로 선택하여 추출하는 방식
: 독립적으로 선택

**단순무작위추출법** : 모든 요소를 목록으로 만들고, 난수를 이용하여 표본을 정하는 것  
**층화추출법** : 모집단을 몇 개의 층(집단)으로 미리 나눈 뒤, 각 층에서 필요한 수의 조사대상을 무작위로 추출하는 방법

$\rightarrow$ 편향된 추출로는 올바른 추정이 어려움
### 데이터 얻는 법
모집단에 대해 추정한 결과를 어느 정도 일반화할 수 있는가는, 각 분야 고유의 지식(도메인 지식)에 따라 달라짐

### 직감적으로 추론통계 이해하기

> **시사점1.** 정말로 알고자 하는 것은 표본 데이터가 아니라 모집단이다.  
> **시사점2.** 모집단의 모든 요소를 다 조사하는 전수조사는 어렵다.  
> **시사점3.** 작은 크기의 표본으로도 모집단을 추론할 수 있다.  
> **시사점4.** 표본을 추출할 때는 무작위로 추출해야 한다.



## 07. 표본오차와 신뢰구간

```
* 신뢰구간의 의미와 활용 방법을 이해한다.
```

### 모집단과 데이터 사이의 오차 고려하기
정말로 알고 싶은 것 = 모집단평균 $\mu$  
모집단의 일부인 크기 n인 표본 x_1, x_2, ...,x_n 을 모집단에서 무작위로 추출하여, 이 표본(데이터)에서 모집단평균 $\mu$를 추정하는 것

### 표본오차(표집오차, sampling error)
‘정말로 알고 싶은 것’과 ‘실제로 손 안에 있는 데이터’에는 어긋남(오차)가 생김

**표본오차는 확률적으로 바뀐다**
표본은 모집단의 성질과 정확히 일치하지 않고, 확률오차를 수반함
따라서 이 ***오차***가 중요

### 큰 수의 법칙(law of large numbers)
표본크기 n이 커질수록 표본평균 $\bar{x}$가 모집단평균 $\mu$에 한없이 가까워진다는 법칙  
-> ***표본평균 $\bar{x} - \mu$ 가 0에 한없이 가까워진다는 뜻***

### 중심극한정리(central limit theorem)
: 모집단이 어떤 분포이든 간에, 표본크기 n이 커질수록 표본평균 $\bar{x}$의 분포는 정규분포로 근사할 수 있음

> **평균: 모집단평균 $\mu$  
> 표준편차: $\frac{\sigma}{\sqrt{n}}$**

### 추정량
: 모집단의 성질을 추정하는 데 사용하는 통계량
> **일치추정량**: n을 무한대로 했을 때 모집단의 성질과 일치하는 추정량  
> **비편향추정량**: 추정량의 평균값(기댓값)이 모집단의 성질과 일치할 때의 추정량  
$\rightarrow$ 매번 얻을 때마다 확률적으로 다른 값이 되지만, 평균으로 보면 모집단의 성질을 과대하지도 과소하지도 않게 나타내는 양  
$\rightarrow$ 표본평균은 모집단 평균의 비편향 추정량  

### 표준오차(standard error)
> 평균: 0   
> 표준편차: $\frac{\sigma}{\sqrt{n}}$

$\sigma$ 는 모집단의 성질이므로 보통 우리로선 알 수 없는 미지의 숫자  
***표본에서 추정한 비편향표준편차 s를 $\sigma$ 대신 사용***  
$\rightarrow$ s / $\sqrt{n}$를 표준오차로 함

### 신뢰구간(cofidence interval)
OO% **신뢰구간**이란?
$\rightarrow$ OO% 의 확률로 이 구간에 모집단평균 $\mu$가 있다  
신뢰구간은 표본에서 구한 모집단 $\mu$의 추정값을 ***어느 정도 신뢰할 수 있는지***를 나타냄

### t분포
: 모집단이 정규분포라는 가정하에 미지의 모집단 표준편차 $\sigma$를 표본으로 계산한 비편향표준편차를 s로 대응했을때, $\bar{x}$ - $\mu$ 를 표준오차 s / $\sqrt{n}$ 로 나누어 표준화한 값이 따르는 분포

> $$t = \frac{\bar{x} - \mu}{s / \sqrt{n}}$$

### 정밀도를 높이려면 표준오차 s / $\sqrt{n}$ 에 주목
이를 작게 하기 위해서는 비편향표준편차 s를 작게 하거나, 표본크기 n을 크게 함
