# 1️⃣ 개념 정리 
## 01. 데이터를 분석하다

```
✅ 학습 목표 :
* 데이터에 대해 이해한다.
* 데이터 분석에 대해 이해한다.
```
### 데이터
: 대상을 관찰하고 측정함으로써 얻을 수 있는 그 대상의 정보  
: 수치의 모음

### 데이터 분석의 목적 - 요약/설명/예측
(1) 데이터를 **요약**하는 것  
: 데이터의 경향을 파악하는 것  
: 가장 흔한 방법은 평균값 사용하는 것

(2) 대상을 **설명**하는 것  
: 대상이 가진 성질과 관계성을 명확히 밝히고 이를 이해한다

**실험군**: 실험에서 효과가 기대되는 처리를 한 집단  
**통제군(대조군)**: 실험군과 비교, 대조하기 위한 집단

#### <설명의 수준에 따른 분류> 
**인과관계 - 깊은 이해**  
: 2가지 중 하나(원인)을 변화시키면, 다른 하나(결과)도 바꿀 수 있는 관계  
*"~하면 ~이 된다"*  
**상관관계 - 얕은 이해**  
: 한쪽이 크면 다른 한쪽도 큰(또는 한쪽이 크면 다른 한쪽은 작은) 관계  
: 한쪽을 변화시켰다 하더라도 다른 한쪽이 변한다고 단정할 수 없음 


(3) 새로 얻을 데이터를 **예측**하는 것  
: 이미 얻은 데이터를 기반으로, 이후 새롭게 얻을 데이터를 예측하는 것  
ex. 선형관계: 사람이 다루고 해석하기 쉬움  
cf. 기계학습 12장

## 03. 통계학의 전체 모습
```
✅ 학습 목표 :
```

### 추론통계
: 수집한 데이터로부터 데이터의 발생원을 추정하는 방법  
: 전체 집단(모집단)에서 표본 추출하여 전체를 추론하는 방법

### 확률 모형

### 통계적 추론  
: 데이터에서 가정한 확률 모형의 성질을 추정하는 방법
### 가설검정
: 세운 가설과 얻은 데이터가 얼마나 들어맞는지를 평가하여, 가설을 채택할 것인가를 판단하는 방법

## 04. 데이터 분석의 목적과 알고자 하는 대상

```
✅ 학습 목표 :
* 데이터 분석의 목적에 맞게 ‘알고자 하는 대상(모집단)’을 명확히 정의하는 것이 왜 중요한지 이해한다.
```
데이터 분석의 목적을 정했다면, 알고자 하는 대상을 명확히 하는 것이 중요함  

**예시**  
데이터 분석 목적: 신약의 효과  
알고자 하는 대상: 고혈압이 있는 **모든** 사람



## 05. 모집단

```
✅ 학습 목표 :
* 모집단의 개념을 명확히 설명할 수 있다.
```
**모집단** : 알고자 하는 대상 전체

단, 알고자 하는 대상이 전체일지라도, 실제로 데이터를 얻을 가능성이 없는 요소를 포함한 모집단은 적절하지 않음.  

**'지금 알고자 하는대상이 무엇인지'**, **'무엇을 모집단으로 설정할 것인지'** 가 중요

### 모집단 크기
: 모집단에 포함된 요소(element)의 수

**유한모집단** : 모집단 중 한정된 요소만 포함  
-> 원칙적으로는 모든 요소를 조사하는 것이 가능함  
-> but, 시간과 비용이 많이 들기 때문에 비현실적임    

**무한모집단** : 모집단 중 포함된 요소의 개수가 무한한 것  
-> 원칙적으로 모든 요소를 조사하는 것이 불가능함  

## 06. 모집단의 성질을 알다 

```
✅ 학습 목표 :
* 전수조사와 표본조사의 개념적 차이와 각각의 필요성을 설명할 수 있다.
* 표본에 대해 이해한다.
```
**모집단의 성질**을 알면 대상을 설명, 이해할 수 있고, 예측 가능함  

### 전수조사
: 모집단에 포함된 모든 요소를 조사  
: 유한모집단일 때 사용 가능한 방법 (무한모집단은 불가능)  
: 분석할 데이터 = 모집단  
cf. 기술통계

### 전수조사의 어려움
비용, 시간이 많이 들기 때문에 실현불가능한 경우가 많음 

### 표본조사
: 표본(모집단의 일부)을 이용해 모집단의 성질을 조사

**표본추출** : 모집단에서 표본을 뽑는 것  
cf. 추론통계: 모집단의 일부를 분석하여 모집단 전체의 성질을 추정  
-> 모집단의 성질을 100% 알아낼 수 없음 (불확실성)

### 표본크기
: 표본에 포함딘 요소의 개수  
: 보통 알파벳 *n* 으로 나타냄

**표본크기** vs **표본개수** 혼동주의!

표본크기는 모집단의 성질을 추정할 때의 확실성이나 가설검정의 결과에도 영향을 끼치기 때문에 중요함

## 문제 1.

> **🧚Q. 한 회사가 직원들의 ‘업무 만족도’를 파악하기 위해 설문조사를 실시했다.
전체 직원 1,000명 중 50명에게만 설문을 진행했고, 그 결과 평균 만족도는 4.5점(5점 만점)이었다.
회사는 이를 근거로 “우리 회사 직원들은 대체로 업무 만족도가 높다”고 결론 내리려 한다.**
>
> **이 결론이 타당한지 판단하기 위해 가장 먼저 확인해야 할 것은 무엇인가?**

~~~
1️⃣ 조사 대상 50명이 전체 직원을 대표하는 표본인가?
2️⃣ 만족도의 평균값만으로 결론을 내려도 되는가?
3️⃣ 설문 문항이 객관적으로 구성되었는가?
4️⃣ 이 결과를 모집단(전체 직원)으로 일반화해도 되는가?
~~~


```
1️⃣ 이라고 생각합니다. 데이터 분석의 목적을 명확히 하기 위해서는 알고자 하는 대상인 모집단을 설정하고, 모집단의 특징을 대표할 수 있는 표본을 추출하는 것이 중요합니다. 그러나 50명이라는 표본크기는 모집단(직원 1000명)을 대표하기에는 부족한 숫자라고 생각합니다. 
```
