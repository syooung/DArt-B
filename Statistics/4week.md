**`산점도(scatter plot)`** : 변수를 x축, y축으로 하여 각 값을 2차원 평면 위에 점으로 나타낸 것.

### 상관(correlation)
: 2개의 확률변수 또는 데이터 사이의 관계성을 의미함  
: 상관이 있다고 해서 원인과 결과를 뜻하는 ***인과관계***가 있는지까지는 알 수 X.

### 회귀 또는 회귀분석(regression, regression analysis)
: y=f(x) 라는 함수를 통해 변수 사이의 관계를 공식화하는 것

**`x`** : 설명변수 또는 독립변수  
**`y`** : 반응변수 또는 종속변수


## 02. 상관관계

```
✅ 학습 목표 :
* 피어슨 상관계수의 개념과 특징에 대해 설명할 수 있다.
* 비모수 상관계수에 대해 설명할 수 있다.
* 상관계수의 유의성 검정과 p값·표본크기의 영향을 이해한다.
```

### 피어슨 상관계수 r (Person’s correlation r)
: 2개의 양적 변수 간 관계의 강도를 정량화하는 방법 중 하나

**`양의 상관(positive correlation)`** : x가 커질수록 y도 커지고, x가 작아질수록 y도 작아짐  
**`음의 상관(negative correlation)`** : x가 커질수록 y는 작아지고, x가 작아질수록 y는 커짐

r의 범위는 -1 ≤ r ≤ 1

> **|r| 값에 따라 상관의 강도를 다음과 같이 해석함**  
>> 0.7 < |r| ≤ 1 : 강한 상관  
>> 0.4 < |r| ≤ 0.7 : 중간 정도 상관  
>> 0.2 < |r| ≤ 0.4 : 약한 상관  
>> 0.0 < |r| ≤ 0.2 : 거의 상관없음  


* **상관계수 r 은 선형관계를 나타낸다**   
<주의할 점>  
  * 피어슨 상관계수 r 은 2개 양적 변수의 ‘선형’ 관계성 강도를 정량화한 것 $\rightarrow$ ***따라서 비선형관계를 정량화할 수 없음***  
  * 직선의 기울기 크기는 관계가 없음


* **상관계수가 같은 다양한 데이터**  
: 같은 r 값을 가지고 있더라도, 비선형을 포함해 다양한 패턴이 나타남  
: 반드시 전형적인 우상향 산점도가 아님 !!

* **정규성 검사**  
: 피어슨 상관계수 r 은 모수적인 방법이므로, ***정규분포***라고 가정함  
: 따라서 정규성을 `샤피로-윌크` 검정으로 확인하고, 정규성이 없다면 **비모수 상관계수** 이용


### 비모수 상관계수
: 데이터의 x축, y축 중 적어도 하나 이상에 **정규성이 없을 때** 사용

**`스피어만 순위상관계수 ρ`**  
**`켄달 순위상관계수 τ`** : 표본크기 n 이 매우 작을 때 권장

### 상관관계 사용 시 주의할 점
2개 변수가 처음부터 종속 관계일 때 주의 필요.  
따라서, x축과 y축의 값이 ***개별 변수***일 것, 그리고 나눗셈 등으로 변환하지 않았을 것을 확인

### 상관계수와 가설
**[ 상관계수의 가설검정 ]**

**모집단**의 상관계수 $r_p$
**표본**의 상관계수 r

  * 모집단에서 두 변수 간에 아무런 관계가 없다고 가정, $r_p$ = 0(무상관)
  * 표본을 뽑아 상관계수 r을 계산해보면
  * **순전히 우연에 의해** 꽤 높은 상관관계가 나오는 경우가 발생.

따라서 **가설검정 필요** 
p값 < 0.05 라면 “양의 상관관계가 있다” 라고 주장 가능

**[ 표본크기와 가설검정 ]**

표본크기 n 이 클수록 p값이 작아져 p < 0.05 이므로 주의해야함.  
따라서, p < 0.05 라고 해서 곧바로 상관이 있다고 판단하는 것이 아니라,
***r값 자체에 눈을 돌려 그 크기를 해석할 필요***가 있음

### 비선형상관

`피어슨 상관계수 r` $\rightarrow$ 양적 데이터의 선형관계  
`스피어만 순위상관계수 ρ` $\rightarrow$ 순위로 변환한 값의 선형관계

**`비선형상관`** $\rightarrow$ ***정보량***에 기반  
‘ X가 Y에 관해, 또는 Y가 X에 관해 어느 정도의 정보를 포함하는지’의 관점에서 관계성 정도를 정량화.

## 03. 선형회귀
```
✅ 학습 목표 :
* 최소제곱법의 원리를 이해한다. 
* 회귀계수의 가설검정에 대해 설명할 수 있다.
* 결정계수에 대해 설명할 수 있다.
* 오차의 등분산성과 정규성에 대해 설명할 수 있다.
* 설명변수와 반응변수에 대해 설명할 수 있다.
```
### 회귀분석이란?
**회귀식** : 설명변수(독립변수) x 와 반응변수(종속변수) y 사이에 y=f(x) 라는 함수를 적용시키는데, 이를 회귀식이라 함

**회귀계수(regression coefficient)** : 회귀식 f(x)의 형태를 결정하는 파라미터 a, b

**회귀모형(regression model)**   
y = a + bx + $\epsilon$

> **회귀분석을 실행할 때 중요한 점**
>> - 어떤 회귀식을 적용할 것인가?
>> - 어떻게 회귀식을 데이터에 적용할 것인가?
>> - 얻은 회귀모형을 어떻게 평가할 것인가?

> **선형회귀**
> : 회귀분석에서 사용하는 회귀식이 ’파라미터에 관한’ 1차식이 될 때 이를 선형회귀라 부름

### [ 최소제곱법 (least squares) ]
: 데이터와 모형 차이의 제곱을 모두 더한 값 $E$를 최소화하는 방법

*좋은 회귀모형이란 데이터에 가능한 한 들어맞는 것*  
즉, ***‘데이터와 회귀식의 차이가 가능한 한 작은‘***

**`확률오차` $\epsilon$**   
: 모든 데이터가 회귀직선 위에 있는 것이 아님. 이를 표현하는게 $\epsilon$  
: $\epsilon$은 반응변수(종속변수) y 에 포함

### 회귀계수

**[ 최량선형비편향추정량 ]**  
: `최소제곱법`으로 얻은 추정량은 비편향추정량 중에서도 가장 정밀도가 높은(분산이 작은) 비편향추정량이 됨

**[ 회귀계수의 가설검정 ]**  
회귀계수를 대상으로 가설검정을 시행할 수 있음

> 귀무가설 : 기울기 b = 0  
> 대립가설 : 기울기 b ≠ 0

가설검정 결과 p < 0.05 라면, 통계적으로 b가 0 이 아니라고 주장할 수 있음  
따라서 "x 에 대해 기울기 b 인 선형관계" !

**`[ 95% 신뢰구간 ]`**
모집단의 회귀계수를 추정하면 ***신뢰구간*** 을 얻을 수 있음

**`[ 95% 예측구간 ]`**
추정한 회귀모형을 기반으로 **데이터 그 자체**가 분포하는 구간을 그릴 수 있는데, 이를 ***예측구간***이라 함

> `신뢰구간` : 모형의 파라미터, 즉 모집단의 범위  
> `예측구간` : 얻을 수 있는 데이터의 범위


### 결정계수 $R^2$ (coefficient of determination, R - squared)
: 회귀식이 잘 들어맞는지 평가하는 지표

But, 결정계수 $R^2$ 은 설명변수(독립변수)의 개수가 늘어날수록 커지는 성질이 있음  
따라서 설명변수 개수 k 에 따라 조정한 조정 결정계수 $R'^2$을 사용


### 오차의 등분산성과 정규성

오차항이 정규분포를 따르고 있는지 여부는, 잔차(데이터와 모형의 차이)의 정규성을 `샤피로-윌크 검정` 을 통해 확인하면 알 수 있음

오차항의 등분산성을 확인하려면, **`브루쉬-페이건 검정(BP 검정)`** 이용  
: 설명변수 x 가 변할 때 잔차 $\hat{y_i} - y_i$ 의 분산이 달라지는지 확인

### 설명변수(독립변수)와 반응변수(종속변수)

1.	한쪽 변수로 다른 한쪽 변수를 **설명**하고자 할 때  
: 설명하는 쪽을 `설명변수` 로, 설명할 대상을 `반응변수` 로 설정.

2.	**인과효과**를 알고 싶을 때  
: 원인을 `설명변수`로, 결과를 `반응변수` 로 설정.  
: 설명변수를 무작위로 할당한 개입 실험에서 얻은 데이터나, 상정할 수 있는 다른 요인도 포함한 다중회귀모형 등을 이용하는 것이 필요

3.	데이터를 **예측**하고 싶을 때  
: 예측의 근거가 될 변수를 `설명변수`, 예측하고자 하는 변수를 `반응변수`로 설정.  
: 회귀모형에 새롭게 얻은 설명변수 값을 대입하여
반응변수 값을 예측 가능  
*단, 예측에 특화된 모형은 언제든 해석 가능한 것은 아님*


## 04. 선형회귀 원리의 확장

```
✅ 학습 목표 :
* 다중회귀를 이해하고 그 결과를 읽을 수 있다.
* 편회귀계수의 의미를 이해하고 해석할 수 있다.
* 범주형 변수를 더미변수로 변환하는 방법과 회귀계수의 해석을 이해한다.
* 공분산분석에 대해 설명할 수 있다.
* 다중공선성에 대해 이해한다.
```
### 다중회귀
: 설명변수가 여러 개인 것

**다중선형회귀모형** 설명변수가 k개
$$y = a + b_1x_1 + b_2x_2 + … + b_kx_k + \epsilon$$

**편회귀계수(partial regression coefficient)** : 기울기 `b`

[ 다중회귀 결과를 읽는 방법 ]  
$\rightarrow$ 편회귀계수와 그 유의성
$R^2$ 값과 마지막의 F통계량에서 얻은 p값에 주목.

### 편회귀계수

**[ 표준화편회귀계수 ]**  

: 회귀분석에서 구한 편회귀계수는 설명변수의 데이터 퍼짐 정도나 단위에 따라 크게 달라지기 때문에 편회귀계수끼리 비교할 수 없음  
: 회귀분석을 시행하기 전에 각각의 **설명변수를 평균 0, 표준편차 1로 변환**한 다음, 회귀분석을 시행하여 구한 회귀계수

**[ 편회귀계수의 해석 ]**  
상관계수가 1에 가까운 강한 상관이 있을 때는, ***다중공선성***이 있는지를 의심, 대처

### 범주형 변수를 설명변수로

**가변수(dummy variable)**   
: 범주형 변수 *ex. 예/아니오, 실험군/대조군, 혈액형* 에는 **대소관계가 없으므로** 0과 1같은 가변수를 설명변수로 이용.

* **범주가 3개 이상일 때**  
ex. 범주가 4개인 혈액형 A, B, O, AB  
  * 0 또는 1인 가변수를 `(범주 개수 - 1)`개 준비
  * `(범주개수 - 1)`인 이유는 ***다중공선성을 피하기 위해서***

### 공분산분석(ANCOVA, analysis of covariance) 
: 결과에 영향을 줄 것 같은 '외생변수'의 효과를 통계적으로 제거한 뒤, 진짜 관심 있는 집단 간의 차이를 비교하는 것

**공변량** : 새로 추가한 양적 변수

> < 공분산분석 사용 전제조건 >
>> - 집단 간 회귀의 기울기가 서로 다르지 않을 것 = 회귀계수가 같을 것  
>> - 회귀계수가 0이 아닐 것.


### 다중공선성(multicollinearity)
: 다중회귀에서 설명변수(독립변수) 사이에 강한 상관이 있는 것  

**$\rightarrow$** 다중공선성이 있으면 추정값의 신뢰성이 ***떨어짐***

### 다중공선성 측정 방법 - 분산팽창인수 VIF(variance inflation factor)

$$ VIF_i = 1 / 1- R_i^2$$

**`VIF > 10`** : 2개 사이의 상관이 아주 강한 것

다중공선성이 강하다고 판단했을 때는,  
1개를 없애거나, 주성분분석 등 차원축소방법 이용하여 설명변수의 개수를 줄임

# 2️⃣ 확인 문제

## 문제 1.

🧚 **Q. 다음은 다중회귀분석 결과이다.**

한 연구자가 학생들의 공부시간(X₁), 수면시간(X₂), 학원 수강 여부(0=미수강, 1=수강)를 설명변수로 하여 시험 점수(Y)를 예측하는 다중회귀모형을 분석하였다.

분석 결과는 다음과 같다.

- 공부시간의 회귀계수: **2.5** (p < 0.01)  
- 수면시간의 회귀계수: **-0.8** (p = 0.20)  
- 학원 수강 여부의 회귀계수: **5.0** (p < 0.05)
